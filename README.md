# 이선우 | 백엔드·AI 개발자

도메인 지식을 신뢰할 수 있는 소프트웨어로 구현하는 백엔드·AI 개발자입니다.

근거를 추적할 수 있는 AI 서비스와 복잡한 업무 규칙을 명확한 코드로 풀어내는 일에 관심이 많습니다. 최근에는 한국어 공공데이터 표준을 검색하는 RAG 시스템과 급여 업무 전 과정을 다루는 ERP를 개발했습니다.

## 주요 프로젝트

| 프로젝트 | 주요 내용 | 기술 스택 |
| --- | --- | --- |
| [한국어 지식 기반 RAG 어시스턴트](https://github.com/sunwoo8478/korean-chatbot) | 정확 일치 검색, 벡터 검색, 리랭킹, 스트리밍 응답, 문서 수집, 관리자 도구를 결합한 한국어 RAG 서비스 | Python, FastAPI, PostgreSQL/pgvector, React |
| [PayFit ERP](https://github.com/sunwoo8478/ERP) | 직원·근태·급여 계산·법정 공제·명세서·리포트·임직원 포털을 연결한 HR·급여 시스템 | Kotlin, Spring Boot, PostgreSQL, React, TypeScript |

## 관심 분야

- 검색 증강 생성(RAG)과 검색 품질 개선
- 백엔드 API 설계와 도메인 모델링
- 기업 업무 프로세스 자동화
- 관측 가능하고 운영하기 쉬운 서비스
