# Lee Sun-woo

Backend & AI engineer focused on turning domain knowledge into reliable software.

I build data-grounded AI assistants and business systems with clear domain models, observable APIs, and practical user interfaces. My recent work spans RAG pipelines for Korean public-data standards and end-to-end payroll workflows.

## Selected projects

| Project | What I built | Stack |
| --- | --- | --- |
| [Korean Knowledge Assistant](https://github.com/sunwoo8478/korean-chatbot) | A Korean RAG assistant that combines exact term lookup, vector retrieval, reranking, streaming responses, document ingestion, and an administration console. | Python, FastAPI, PostgreSQL/pgvector, React |
| [PayFit ERP](https://github.com/sunwoo8478/ERP) | A payroll and HR platform covering employee data, attendance, payroll calculation, statutory deductions, payslips, reporting, and an employee portal. | Kotlin, Spring Boot, PostgreSQL, React, TypeScript |
