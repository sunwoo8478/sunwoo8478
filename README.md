## 작업 방식

- 기능을 구현하기 전에 데이터 흐름과 책임 범위를 먼저 정리합니다.
- 복잡한 업무 규칙은 이름이 드러나는 작은 단위로 나눕니다.
- README에 실행 방법과 설정 조건을 남기고 CI에서 기본 검증을 반복합니다.
- 검색 결과와 자동 생성된 답변은 근거를 확인할 수 있도록 구성합니다.
