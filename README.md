## 제가 집중하는 것

- **검색 증강 생성(RAG)** — 검색 정확도, 리랭킹, 출처 추적이 가능한 답변
- **백엔드 설계** — 읽기 쉬운 도메인 모델과 유지보수하기 좋은 API
- **업무 자동화** — 복잡한 기업 업무를 실제로 사용할 수 있는 서비스로 구현
- **안정적인 운영** — 테스트, CI, 관측 가능성을 고려한 개발

## 🚀 대표 프로젝트

### [한국어 지식 기반 RAG 어시스턴트](https://github.com/sunwoo8478/korean-chatbot)

> 한국어 공공데이터 표준을 정확하고 빠르게 탐색하는 지식 검색 서비스

- 정확 일치 검색과 벡터 검색을 결합한 하이브리드 검색
- 리랭킹, 스트리밍 응답, 문서 수집 및 관리자 도구 구현
- 답변 근거를 확인할 수 있는 출처 중심의 RAG 파이프라인
- `Python` `FastAPI` `PostgreSQL` `pgvector` `React`

### [PayFit ERP](https://github.com/sunwoo8478/ERP)

> 직원·근태·급여·법정 공제·명세서를 하나로 연결한 HR·급여 시스템

- 급여 계산과 법정 공제 규칙을 도메인 로직으로 구현
- 관리자와 임직원 포털, 리포트 및 명세서 흐름 통합
- 백엔드 테스트와 프론트엔드 빌드를 자동 검증하는 CI 구성
- `Kotlin` `Spring Boot` `PostgreSQL` `React` `TypeScript`
